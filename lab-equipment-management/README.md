# 实验室设备借用管理系统

这是一个基于 Vue 3 + TypeScript + Vite + Element Plus 的实验室设备借用管理系统前端项目。

## 功能特性

- **用户身份**：支持学生和管理员两种角色模拟登录。
- **设备浏览**：查看实验室设备列表，支持按状态（可借用/已借出）筛选。
- **借用申请**：学生可以填写借用时间、用途并提交借用申请。
- **个人中心**：学生查看自己的借用记录，支持归还操作。
- **设备管理**（管理员）：添加、编辑、删除设备信息。
- **借用审核**（管理员）：审核学生的借用申请，标记设备归还状态。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia (状态管理)
- Vue Router (路由)
- Element Plus (UI组件库)

## 项目运行

### 安装依赖

```sh
npm install
```

### 启动开发服务器

```sh
npm run dev
```

### 构建生产版本

```sh
npm run build
```

## 默认账号说明

系统采用模拟登录，输入任意姓名即可登录。

- **学生角色**：选择"学生"身份登录。
- **管理员角色**：选择"管理员"身份登录。
